# 코틀린 기본 문법

## 자료형

자바와 달리 자료형을 추론 > 타입을 생략 가능하다.

## 변수와 상수

`var` - variable : 변수  
`val` - value : 상수

## 연산자

연산자의 경우 자바와 동일하다.

## 조건식

`if-else`는 자바와 동일

+ when
    ```kotlin
    when (변수 또는 상수) {
        값 -> {
  
        }
        값2 -> {
  
        }
        in 값3..값9 -> {
  
        }
        else -> {
  
        }    
    }
  ```

## 반복문

`while`문은 자바랑 동일하다.

+ for
    ```kotlin
    for (요소 in 리스트) {
        소스코드 (로직)
    }

    // 숫자의 범위로 반복
    // 시작값 이상 ~ 마지막값 이하
    for (인덱스 in 시작값 .. 마지막값) {
        소스코드 (로직)
    }
  
    // 시작값 이상 ~ 마지막값 미만
    for (인덱스 in 시작값 until 마지막값) {
        소스코드 (로직)
    }
    ```
